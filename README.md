# Chess over SSH

## Tech Used
1. Golang
2. Bubble Tea
3. Wish

## How to run
To start the SSH server that will server a bubble tea chess application run 
```shell
make setup
make run
```
SSH will start and to connect to is server
1. Open a new terminal
2. ssh -p 23234 localhost
3. Start 2 instance for testing
